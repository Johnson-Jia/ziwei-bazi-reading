# 04 · 四化星 · 倪师原话

## 四化定性（天纪 03 倪师原话）

### 化科
- **主科名**（参加考试，金榜题名）
- 文章写得好，文笔很好
- 画画好，才艺出众
- **代表名气很大**
- 身体特征：手掌有天柱纹（一路通天）
- 气色：黄润（像黄玉）

### 化权
- **代表官印**（当官的印）
- **代表性刚**（个性很刚，天生是领导者）
- 身体特征：颧骨要有肉；犀牛骨突出
- 忌孤权皮骨相连

### 化禄
- **禄就是财禄、钱财**
- 命宫化禄的人可能是小气鬼、守财奴（「性守」）
- 身体特征：
  - 男人鼻子（狮鼻、悬胆鼻为大富）
  - 女人手枯干
- 气色：眼睛眯眯（赚大钱时开心得眯起来）

### 化忌
- **化忌代表劫杀**
- **化忌代表想不开**
- 「化忌最不好了...想把这个忌给你擦掉，但没有办法，擦掉就没有警告」
- 身体特征：脸很大但五官挤在一起、皱在一起 = 一辈子想不开
- 气色：晦暗、灰尘色，像没洗脸

## 四化运行规则（重要·倪师原话，天纪 03）

> **「四化有科、权、禄、忌。我们教诸位排出来以后，就永远固定在宫上不要动...不需要你每年四化都在不同的位置，这个都不对的。我们每年呢，是流年在动，星都不动。」**

**倪师体系的四化规则**：
- 本命四化：按出生年干推，**永久固定**在盘上
- 流年运势：是**宫位动**（流年入不同宫），**不是星重新化**

这与北派飞星（每个大限 / 流年宫干重新推化）有微妙差别，需要注意：
- 倪师《天纪》讲的是**"本命四化 + 流年宫位"**为主
- 现代北派四化派（钦天 / 梁氏）才严格强调**每一限每一年都按宫干重新起四化**

⚠️ **项目 db-analysis.ts 里的"大限宫干四化"是北派共识，并非严格倪师《天纪》口述**。
    需要标注：此为"北派飞星 × 倪师体系融合"而非单纯倪师原话。

## 组合原则

- **因科而禄**：三方四正只有科时 → 因科而禄 → 有专业技术专长（天纪 11 原话）
- **权禄相逢**：自己做事业当老板（天纪 11 原话）
- **科权禄全在大限**：可大富贵，**限好比命好还佳**（天纪 03 原话）
- **化忌所在宫**：该星所主领域受阻 / 想不开 / 是非

## 倪师关于四化的"看盘顺序"

从天纪 03 + 天纪 12 原话归纳：

1. **先看科、权、禄、忌** — 决定格局底色
2. **官禄宫最喜欢权** — 掌权
3. **财帛宫最喜欢禄 / 武曲 / 太阴** — 生财
4. **命宫化忌** — 最需注意的警讯
5. **迁移宫化忌（对冲命宫）= 半空折翅** — 中年危险
