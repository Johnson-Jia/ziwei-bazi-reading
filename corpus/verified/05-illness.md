# 05 · 疾厄宫 · 倪师原话（重大修正）

## 核心修正

> ⚠️ **倪师疾厄论断以"宫位（地支）→ 脏腑"为主，不是"星曜五行 → 脏腑"**

项目原 `db-analysis.ts` 里的 `ZIWU_LIUZHU` 按星曜五行映射脏腑是**偏离倪师体系**的，需要重建为**宫位地支 → 脏腑** 的映射。

## 倪师的宫位 → 脏腑映射（天纪 05 案例提炼）

| 地支宫 | 脏腑 / 部位 | 倪师出处 |

**倪师实际在天纪 05 原话提到的**：
- 化忌在**酉宫** = 肾
- 火星、地空在**丑宫** = 肝
- 疾厄宫**头肩胸**区域 = 妇科子宫、右边卵巢

## 煞星入疾厄

- **擎羊、陀罗入疾厄** = 开刀见血光（天纪 11、12）
- 铃星 + 陀罗 + 化忌落陷 = 中医观念上很惨（天纪 06）

## 身体部位 / 大限年龄对应

- 早年大限（3-12 岁）在命宫代表头部
- 12-22 岁代表颈胸
- …（12 宫顺时针对应身体由上至下，传统紫微五马图）

## 星曜 × 脏腑（仅作辅助，非主轴）

倪师偶尔会提：
- 太阴化忌 → 母亲 / 妇科
- 武曲化忌 → 动刀（手术）
- 破军 → 先天体质薄弱

但**主判还是以宫位**为主，星曜只是"加重 / 减轻该宫位的脏腑问题"。

## 结论

**疾厄宫分析应该按以下顺序**：
1. **主轴**：疾厄宫**所在地支宫** → 脏腑部位
2. **加重**：该宫有无煞星（羊陀火铃）、化忌 → 风险等级
3. **辅助**：疾厄宫主星的五行提示
4. **时间**：大限 / 流年飞到该宫 → 发病时间
