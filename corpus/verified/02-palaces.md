# 02 · 十二宫 · 倪师原话

## 命宫 vs 身宫

- 命宫：先天命，决定这辈子的**特点**
- 身宫：**后天**；身宫落哪里决定人生方向
  - 身宫在财帛 → 私企 / 自己做老板
  - 身宫在官禄 → 做官
  - 身宫在迁移 → 外地发展
  - 身宫在夫妻 → 感情决定人生方向
- 出处：天纪 05、天纪 12

## 官禄宫

- **最喜欢权星入进来**
- 化权或紫微七杀同宫 = **官权加重，非常好**
- 财星跑到官禄里 = **贪官**
- 七杀 + 擎羊 = 当官势必不好
- 官带星越大官做得越大；越小官就越差
- 出处：天纪 05、天纪 08

## 财帛宫

- 太阴入财帛 = 正财
- 武曲入财帛 = 大财星
- 紫微入财帛 = 守成为主（不主动生财）
- 禄存入财帛 = 守财能力强（小气鬼）

## 迁移宫（命宫对宫）

- 紫微在迁移 = 外地会逢贵人，但也是孤君象
- **迁移宫化忌 = 半空折翅**（倪师原话，天纪 06 第23-32大限案例）
- 日月反背在迁移 = 离祖外乡发展

## 夫妻宫

- 需看三方四正，科权禄会合才好
- 右弼孤星独守夫妻宫 = **二婚必离**
- 化忌对冲夫妻宫 = 夫妻死别风险
- 出处：天纪 07、天纪 10

## 子女宫

- 空宫时要看对宫，**对宫化忌则无子**
- 出处：天纪 05

## 疾厄宫（按宫位·地支推脏腑）

**⚠️ 关键修正**：倪师疾厄论断**不是**按星曜五行，而是按**宫位地支**。

- 化忌在酉宫 = 肾
- 火星 / 地空在丑宫 = 肝
- 头肩胸 = 妇科子宫，右边卵巢
- 陀罗 / 擎羊入疾厄 = **开刀见血光**
- 疾厄宫陀罗 + 铃星 + 化忌落陷 = 中医观念上很惨
- 出处：天纪 05、天纪 06、天纪 11

## 福德宫（**女命极其重要**）

- 倪师原话：「**女人的命，福德宫非常重要！福德宫等于讲的也是夫妻**」
- 福德宫廉破 = **横祸、意外灾、夫妻生离死别**
- 水中作冢（福德廉破）= 极凶
- 出处：天纪 09

## 父母宫

- 左辅或右弼**单星独守**父母宫 = 父母离散
- 太阴化忌 = 母亲担心
- 吉星落陷 = 六亲薄
- 出处：天纪 11

## 兄弟宫

- 兄弟宫化忌 = 灾星集中在六亲方面，需独立自强
- 出处：天纪 12

## 田宅宫

- 紫微入田宅 = 祖产有规模
- 太阴入田宅 = 田宅主（倪师未在已抓集数原话强调，传统认知）

---

## 倪师十二宫的核心观点

「十二宫代表一个人被外面十一个层面的影响」——你就是命宫，其余十一宫都是外界对你的作用力。
