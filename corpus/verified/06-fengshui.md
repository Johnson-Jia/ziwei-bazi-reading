# 06 · 阳宅化命 · 倪师原话（天地人三才之"地"）

> 倪师天纪的命盘分析永远不只停留在命理层，必定结合**阳宅布局**做化解

## 核心原则

- 「**位等于名，这是万事兴；名等于位，这是家和**」（天纪 13）
- 住在与角色相符的方位，强化该角色的正面表现
- 「一念之间、一个位置的改变，能够转变人的想法，进而改变命运走向」（天纪 12）

## 常见化命方案（倪师原话）

### 女命阳宅调整
- 37 岁前未婚：住**东南角**（长女位置，催婚）— 天纪 09
- 41 岁后进入好大限再结婚
- 已婚女性主卧在**西北角**很重要（天纪 09）
- 离婚后住西北角 = 「母代父职，完了，终生不嫁」（天纪 12）
- 再婚带孩子 → 住**西南角**（母亲 + 老板位） — 天纪 11

### 无子宅（倪师原话 · 天纪 05）
1. 厕所在房子中间 → 主死先生，**死心脏病、中风、气喘**
2. 西北角是厨房 → 主**夫妻死别**
3. 东边床压墙壁 → 生女儿
4. 床靠北或西 → 生儿子

### 怀孕位置
- 行房位置 → 怀孕只是暂时（东 / 北生男，东生女）
- 出处：天纪 05

## 命 + 阳宅联动

> 「命中没有儿子 → 让他住到南方去，水火既济卦...随便什么位置，我们用卦来变化」

即命盘死定的事，可以通过改变地理位置激发不同的卦，产生不同结果。

## 疾厄 × 阳宅

- 厕所位置错误 → 直接影响心脏 / 呼吸道
- 厨房在西北 → 直接影响夫妻关系
- 次子住西南 → 婚姻延后（水地比）
- 妈妈住正北 → 性格凶悍（地水师）

## 应用到 AI 解读

在**健康 / 感情 / 家庭**话题下，若命盘有重大风险，应提示：
- "倪师建议可通过调整阳宅化解。例如：XXX"
- 不强制推销风水，但要**让用户知道有化解之道**
- 呼应「命不是人生的全部 → 天地人三才」的原则
