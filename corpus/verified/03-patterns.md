# 03 · 格局 · 倪师原话

> 规则：每个格局要标明「倪师原话出处」或「倪师实际在哪一集引用过」

## 必定成格

### 日丽中天
- 太阳在午宫守命（入庙）
- **武职大利，参谋总长以上**
- 出处：天纪 08

### 日月并明
- 太阳入庙 + 太阴入庙 同时
- 一生公正廉明，贵人多，名利双全，**做事情左右逢源，一辈子做事情荣华**
- 出处：多集

### 紫府同宫 / 紫府朝垣
- 同宫 = 福禄双全
- 朝垣 = 三方有紫微天府同时会照命宫
- 出处：天纪 07

### 雄宿朝元
- 紫微辰戌宫 + 廉贞寅申独守 + 贪狼对宫
- 出武官 / 商人（商人后期毁在桃花）
- 出处：天纪 07

### 明珠出海
- 紫微午宫 + 太阳寅 + 太阴亥 + 命宫未
- 日月并明的特殊配置
- 出处：天纪 07

### 七杀朝斗
- 七杀寅申宫守命 + 对宫紫微天府
- **爵禄荣昌**
- 出处：天纪 07

### 英星入庙
- 破军子宫或午宫守命
- 男命：发武官
- 女命：孤单，瘦瘦干干，晚婚
- 出处：天纪 10

### 火贪格 / 铃贪格
- 贪狼 + 火星 / 铃星同宫
- **出将入相，武贵之路**，偏财暴发
- 出处：天纪 04、天纪 05

### 机月同梁
- 天机 + 太阴 + 天同 + 天梁三方会照
- 文质彬彬，适合公职、学术、服务业
- **（传统共识，倪师已抓集数未直接强调，保留记号）**

### 杀破狼
- 七杀、破军、贪狼三方会照
- 开创格，一生跌宕起伏，大起大落
- 武官配置最佳
- 出处：天纪 04

### 廉贞天相
- 同宫主印绶，清廉
- **（传统共识，已抓集数未单独讲）**

## 凶格

### 半空折翅
- 命宫在亥，对宫廉贞贪狼落陷（或其他迁移化忌对冲命宫）
- **天罗地网，中年限（30岁左右）有夭折现象**
- 出处：天纪 06（23-32大限案例原话：「廉贞贪狼，半空折翅，很危险」）

### 自杀格
- 廉贞破军在亥宫
- 出处：天纪 07

### 地劫孤星独守
- 比与其他星同宫危害更大
- 出处：天纪 08

### 羊陀夹命 / 火铃夹命
- 煞星夹命，刑克 / 性急刑伤
- **（传统口诀，倪师语境里多次用「刚硬」「急」表达，但未必这两句原话）**

### 昌曲化忌
- 文书暗亏、文书契约是非
- **（传统口诀）**

## 倪师关于格局的方法论

1. **「格一样，宫不一样，批起来是相反的」** — 同样星曜组合落不同宫，含义完全相反（天纪 06 原话）
2. **「命好不如限好」** — 命普通但大运有科权禄也可大富贵（天纪 03 原话）
3. **「官带星越大官做得越大」** — 正位为先（天纪 03 原话）
4. **「先看科、权、禄、忌」**（天纪 12 原话）
5. **「三方四正里只有科 = 因科而禄，有专业技术专长」**（天纪 03、11 原话）
6. **「命相同参会增加准确性」**（天纪 03 原话）
