## 三、十神体系（以日干为"我"）

口诀：**生我父母（印），我生子孙（食伤），我克妻财（财），克我官鬼（官杀），同我兄弟（比劫）。**

| 关系 | 同性（阳阳/阴阴） | 异性（阴阳） |
|---|---|---|
| 生我 | 偏印（枭神） | 正印 |
| 我生 | 食神 | 伤官 |
| 我克 | 偏财 | 正财 |
| 克我 | 七杀（偏官） | 正官 |
| 同我 | 比肩 | 劫财 |

> 判定技巧：与日干**阴阳相同**者为偏（比肩/食神/偏财/七杀/偏印）；**阴阳相异**者为正（劫财/伤官/正财/正官/正印）。
> 白话明示：**阳见阳 / 阴见阴 = 比肩**（甲见甲、乙见乙）；**阳见阴 / 阴见阳 = 劫财**（甲见乙、乙见甲）。

## 四、日主旺衰判断（核心）

判断日干（日主）强弱，看"三得"，但**三者权重不均等，不可机械计数**：

1. **得令（月令）——权重最大（约占全局半数影响力）**：日主在月支所处十二长生状态。生扶日主的月令（如甲木生于亥子水月或寅卯木月）为得令旺；克泄耗月令为衰。**得令与否常是旺衰第一决定因素**，但得令而无根无帮仍可能身弱，失令而地支强根成群亦可能身旺——须结合地、势综合。
2. **得地（地支有根）——须区分根的强弱**：地支藏干中有与日主同类或生扶之五行。**本气根（如甲在寅卯、禄旺位）力量最强；长生/墓库根次之；余气根（如甲在辰中乙木）力量甚弱**，不可与禄旺根等同视之。有强根则稳，仅余气虚浮仍偏弱。
3. **得势（天干帮扶）**：天干多印（生我）比（同我）帮扶则旺，多财官食伤克泄耗则弱。

**综合定强弱**：以**月令为主导**，辅以根之虚实（本气/余气）、天干党势，通盘权衡——**非简单"三者居二即旺"**。

**⚠ 调候优先**：若命局**寒暖燥湿严重失衡**（如金水偏寒、火土偏燥），《穷通宝鉴》体系主张**调候用神优先于旺衰用神**——先取调候之神（丙丁取暖、壬癸降温），旺衰取用让步。此类命局不可执泥旺衰。

**关键点**：身强身弱不是绝对好坏，而是定**喜用神方向**的依据。

## 五、喜用神与忌神（定方向）

| 日主 | 喜用神（帮身生身） | 忌神（克泄耗身） |
|---|---|---|
| **身旺** | 官杀（克）、食伤（泄）、财（耗） | 印（生）、比劫（帮） |
| **身弱** | 印（生）、比劫（帮） | 官杀（克）、食伤（泄）、财（耗） |

**特殊格局（须严格判定，不可滥立）**：
- **从强 / 从弱**：顺势为用（从强喜印比泄秀、从弱喜财官食伤）。**成立严格前提**：日主**全无通根**、全局**无一丝生扶（从弱）或无一丝克泄（从强）**、五行偏枯至极。**普通偏弱/偏旺命局绝不可论从**——误判从格会导致用神完全取反。拿不准时按普通身弱/身旺论更稳妥。
- **专旺 / 稼穑 / 曲直等**：五行一气专旺成格，顺其势为用，最忌克。

### 常见格局速查（均含前提与例外，忌绝对化）
- **财多身弱**：财星旺而日主弱，"富屋贫人"，强行求财必破。**例外**：若有**印比帮身**（印生身、比劫助身）则能担财，格局反转为富。
- **身旺财旺**：能担财，主富。**反面**：身旺而无财官食伤泄耗，反主孤独贫乏（身旺无依）。
- **官杀混杂**：正官七杀同现，主事多变、压力、是非。**例外**：身旺用杀、伤官制杀、印星化杀，反成贵格（"杀印相生""伤官驾杀"）。
- **伤官见官**：伤官与正官相克，主口舌波折。**例外**：《滴天髓》明示**金水伤官、木火伤官**等调候所需之官，或有财通关、有印护身时，伤官见官反显贵——不可一概断凶。
- **印星化官杀**：官杀旺而有印化解，主以学识/技术化解压力，贵格。**反面**：印过重而身已旺，反主滞塞孤克（母慈灭子）。
- **食神生财**：食神与财顺生，主以技艺得财。**反面**：食伤泄身过旺而身弱，反主贫困劳碌（泄气太过）。

## 六、大运（每运十年）

### 顺逆规则
- **阳年男 / 阴年女 → 顺排**（从月柱的下一柱起，顺天干地支序）
- **阴年男 / 阳年女 → 逆排**（从月柱的上一柱起，逆天干地支序）

### 起运岁（易错，务必严谨）
从生日数到最近的"节"（非"气"）：顺排数到下一节，逆排数到上一节，**3 天折 1 岁**。

**口径铁律：起运岁一律用虚岁**。精确换算规则：**出生当年即虚岁 1，每过一个农历新年（春节）+1**。

注意"出生后 N 实年"与"虚岁"**并非简单 +1**——取决于出生日距下一个春节的远近：
- 通常出生后第 N 个公历年交运，虚岁 ≈ N+1；
- **若出生在农历年末（接近春节）**，出生后不久即过春节，实年与虚岁差值会变大（可能出现 N 实年 = 虚岁 N+2）。
- 故**务必用专业排盘工具（如问真八字）直接读"X 岁起运 + 交运日期"**，不要手算虚岁换算。

例：命主 1993-10-20 生，逆数到寒露（10/8）得 12.6 天 ≈ 4.2 实年，交运落在 1997 年底～1998 年初；1997 年春节（2/7）后命主虚岁 5，至 1998 年春节（1/28）前仍虚岁 5，故交运时**虚岁 5**，记"5 岁起运"。

**务必用专业排盘工具核对**（如问真八字），避免手算口径/节气时分/计数端点出错。起运岁错了，整条大运分段会整体平移 1 岁、当前大运判断跟着错——但流年逐年分析不受影响（流年按公历干支固定）。

### 大运分析方法
1. 大运天干地支的十神 → 定该十年基调（财运/官运/印运/比劫运/食伤运）。
2. 大运十神是日主**喜用还是忌神** → 喜用运顺、忌神运逆。
3. 大运与四柱的冲合刑害 → 定具体事件。
4. 大运**干支十年全程共同作用**，非割裂：前五年**天干权重更高**、后五年**地支权重更高**，但绝非"前五年不看地支、后五年不看天干"。干支生克、与日主喜忌、与四柱冲合须通盘论。

## 七、流年应吉应凶（八字最硬的依据）

**核心逻辑：流年十神是日主喜用神 → 该年应吉；是忌神 → 该年应凶。** 吉凶方向是确定的，这是八字推流年最有力的工具。

分析步骤：
1. 流年天干地支 → 转十神（相对日干）。
2. 判喜忌（身弱喜印比、身旺喜财官食伤）。
3. 喜用年 → 应学业、贵人、晋升、得财、姻缘（看具体十神）之吉。
4. 忌神年 → 应破财、压力、伤病、口舌、感情波折之凶。
5. 结合大运：大运喜用＋流年喜用＝大吉；大运忌神＋流年忌神＝大凶；一喜一忌＝吉凶参半。

**各十神喜用年应事**：
- 印（喜用）：学业、文凭、长辈贵人、安顿、置业
- 比劫（喜用）：朋友合伙、助身担财、合作得力
- 食伤（喜用）：才华施展、技艺、子女、表达得利
- 财（喜用）：得财、置业、男命姻缘
- 官杀（喜用）：升职、掌权、地位、名声、女命姻缘

## 九、定时辰（用八字反推）

时辰不同 → 时柱不同 → 可能导致：
1. **日主旺衰逆转**（如甲戌时无水印根身弱 vs 乙亥时有亥水印根身旺）
2. **喜用神完全反转**（身弱喜印比 ↔ 身旺喜财官食伤）
3. **同一年吉凶方向相反**

**反推法**：对每个候选时辰，分别推算"某关键流年是喜用年（应吉）还是忌神年（应凶）"，与命主真实事件对照——方向一致则该时辰成立，矛盾则排除。多组方向性验证同时成立即为该时辰。

最强应事维度（八字视角）：
- 父亲：偏财父星，忌财年应父凶（如手术刑伤）
- 学业：印星，喜印年应学业之吉（如得文凭）
- 性格职业：用印＝技术内敛型，用食伤＝口才外向型
- 健康：日主受克耗程度（如甲木受酉金克、土耗＝神经/肝胆弱）
- 财运：财多身弱者强行求财必破（验创业投资成败）
- **婚姻**：男命财（妻）喜忌——喜财年应姻缘之吉、忌财年应婚变，是定时辰高效应事维度
- **格局高低**：两时辰日主旺衰若逆转，身旺能担财官 vs 身弱富屋贫人，人生层次差异显著，可据命主实际成就层次反推
- **早年灾病**：日主受克耗程度不同，幼年体质/灾厄差异明显

## 十、推演步骤

1. **排四柱**：年月日时，列天干、地支、藏干、十神、纳音。
2. **五行力量统计**：数各五行（含藏干）多寡，标旺衰。
3. **定日主旺衰**：三得（令/地/势）综合判断身强身弱。
4. **定喜用忌神**：身弱喜印比忌财官食伤；身旺反之。
5. **定格局**：财多身弱/身旺财旺/官杀混杂等。
6. **排大运**：顺逆＋起运岁，标各运十神与喜忌。
7. **六维归纳**：事业/财/婚/健康/学业/六亲，每维给结论＋十神依据。
8. **流年逐年**：用喜忌应吉应凶推演，验算过往＋预测未来，标喜用年（机遇）与忌神年（防范）。
9. **综合建议**：扬长（用喜用）避短（忌忌神），针对性化解。
10. **免责声明**。

## 十一、置信度标注规范

- 🟢 高：喜用/忌神方向明确且有多流年应事印证（如某忌财年父亲手术＋某喜印年得文凭，双向印证）
- 🟡 中：单一十神或格局推断，需更多应事佐证
- 🔴 低：仅依据暗藏之气或间接推断

每个关键结论后附 `(依据：XX十神/喜忌/格局；置信度🟢/🟡/🔴)`。

## 十三、十维度预测指引（算法→LLM→具体事项）

### 预测四原则（必守）
1. **给事项类别，不给唯一确定事项**（命理预测精度有限：胎停年份月份曾推错）
2. **标概率**（🟠/🟡/⚪），让用户知可信度
3. **作"注意力分配"参考**（凶年防范/吉年把握），非宿命论
4. **同类先例参照**（如2020七杀年胎停→2028七杀年备孕谨慎，概率参照非重复）

## 附录·门派争议备注（本文档采用主流通说；下列分歧排盘时可切换比对，不下单一结论）

- **旺衰判断优先级**：传统书房派重**月令得令**为主；部分盲派重**通根得地**为主。本文以月令主导、三得通盘权衡。
- **大运轻重**：有"重地支""干支并重""天干管七年地支管三年"等不同主张。本文采用主流"干支十年共同作用、前五干重后五支重"。
- **格局取用**：《子平真诠》重**月令格局**（以月令所藏定格取用）；《滴天髓》重**全局平衡**（旺衰调候取用）。本文以平衡法为主，重要命局可两参。
- **子女星**：传统子平男命以官杀、女命以食伤为子女；任铁樵一派无论男女皆以食伤为子女（见第八节）。
- **六亲细分**：偏印/正印谁主母亲、比肩/劫财如何细分兄弟姐妹，不同典籍略有出入，本文取通说。
