> 本文件为 `ziwei-bazi-reading` 技能的八字命理（子平四柱）子方法。**独立体系，不含紫微术语**。调用本文件时，遵循 SKILL.md 的核心铁律（严谨、附依据、必加免责声明）。

从用户处获取：年、月、日、时（最好精确到时辰，真太阳时更佳）、性别、出生地（用于真太阳时换算）。
排出**四柱**（年柱、月柱、日柱、日主所在为日干、时柱）。

## 三、十神体系（以日干为"我"）

口诀：**生我父母（印），我生子孙（食伤），我克妻财（财），克我官鬼（官杀），同我兄弟（比劫）。**

| 关系 | 同性（阳阳/阴阴） | 异性（阴阳） |
|---|---|---|
| 生我 | 偏印（枭神） | 正印 |
| 我生 | 食神 | 伤官 |
| 我克 | 偏财 | 正财 |
| 克我 | 七杀（偏官） | 正官 |
| 同我 | 比肩 | 劫财 |

> 判定技巧：与日干**阴阳相同**者为偏（比肩/食神/偏财/七杀/偏印）；**阴阳相异**者为正（劫财/伤官/正财/正官/正印）。
> 白话明示：**阳见阳 / 阴见阴 = 比肩**（甲见甲、乙见乙）；**阳见阴 / 阴见阳 = 劫财**（甲见乙、乙见甲）。

## 四、日主旺衰判断（核心）

判断日干（日主）强弱，看"三得"，但**三者权重不均等，不可机械计数**：

1. **得令（月令）——权重最大（约占全局半数影响力）**：日主在月支所处十二长生状态。生扶日主的月令（如甲木生于亥子水月或寅卯木月）为得令旺；克泄耗月令为衰。**得令与否常是旺衰第一决定因素**，但得令而无根无帮仍可能身弱，失令而地支强根成群亦可能身旺——须结合地、势综合。
2. **得地（地支有根）——须区分根的强弱**：地支藏干中有与日主同类或生扶之五行。**本气根（如甲在寅卯、禄旺位）力量最强；长生/墓库根次之；余气根（如甲在辰中乙木）力量甚弱**，不可与禄旺根等同视之。有强根则稳，仅余气虚浮仍偏弱。
3. **得势（天干帮扶）**：天干多印（生我）比（同我）帮扶则旺，多财官食伤克泄耗则弱。

**综合定强弱**：以**月令为主导**，辅以根之虚实（本气/余气）、天干党势，通盘权衡——**非简单"三者居二即旺"**。

**⚠ 调候用神是四参合之一（非必"严重失衡"才取）**：月令寒暖燥湿一旦偏颇（金水偏寒、火土偏燥、秋木枯槁、冬水冻结等），《穷通宝鉴》即取调候之神（丙丁取暖、壬癸降温、癸水润根等）参于定用——**不必等到"严重失衡"**。如甲日戌月（秋木枯槁）即需癸润+丁暖，纵然旺衰未至极端亦然。调候用神即便属旺衰之"忌"，亦当以调候所需酌取。详见第五节「四参合定用」。

**关键点**：身强身弱不是绝对好坏，而是定**喜用神方向**的依据。

## 五、喜用神与忌神（定方向）

| 日主 | 喜用神（帮身生身） | 忌神（克泄耗身） |
|---|---|---|
| **身旺** | 官杀（克）、食伤（泄）、财（耗） | 印（生）、比劫（帮） |
| **身弱** | 印（生）、比劫（帮） | 官杀（克）、食伤（泄）、财（耗） |

> 上表是**旺衰一维**的基础喜忌，**非最终用神**。定真用神须再过「格局+调候+流通」三参合（见下），否则易犯"缺五行即忌"的机械误判——如身弱财多而缺食伤者，食伤本为通关补缺之用，却被旺衰误判为忌。

### 定用神须「四参合」（核心·禁只看旺衰）

真用神由 **旺衰 ∩ 格局 ∩ 调候 ∩ 流通** 四者参合而定，冲突时权衡主次：

1. **旺衰用神（基础·平衡法 `[古诀]·滴天髓`）**：身弱喜印比、身旺喜财官食伤（即上表）——定大方向。
2. **格局用神（`[古诀]·子平真诠`·月令取格）**：以月令所藏本气定格（如戌月戊土→偏财格），用神须配合成格（财格身弱用食伤生财或印比扶身；官印相生用印化官等）。
3. **调候用神（`[古诀]·穷通宝鉴`）**：按月令寒暖燥湿取调候之神（见第四节⚠）。**调候用神即便属旺衰忌神，亦当酌取**（如己土寅月需癸润，癸为己之偏财，身弱忌财但调候所需故取）。
4. **流通用神（`[古诀]·滴天髓·源流`）**：观五行源头到流尾是否顺生贯通。**断链处即病处**——如木日主财多（木→土缺火通关，木硬克土），所缺的食伤（火）正是通关之用，由忌转用。

**🔴 缺五行不即忌（铁律）**：原局缺或近缺（藏库不透）的某五行，往往是**补缺、通关、泄秀**所需之用，**不得机械判忌**；须看它是否调候所需、是否通关断链、是否泄秀生财——若是则由忌转用。机械旺衰引擎最易在此误判（项目 `bazi_core.js` 设 `BAZI_SIHE` 开关做四参合复核，默认 on，可 off 对照旧旺衰）。

**特殊格局（须严格判定，不可滥立）**：
- **从强 / 从弱**：顺势为用（从强喜印比泄秀、从弱喜财官食伤）。**成立严格前提**：日主**全无通根**、全局**无一丝生扶（从弱）或无一丝克泄（从强）**、五行偏枯至极。**普通偏弱/偏旺命局绝不可论从**——误判从格会导致用神完全取反。拿不准时按普通身弱/身旺论更稳妥。
- **专旺 / 稼穑 / 曲直等**：五行一气专旺成格，顺其势为用，最忌克。

### 常见格局速查（均含前提与例外，忌绝对化）
- **财多身弱**：财星旺而日主弱，"富屋贫人"，强行求财必破。**例外**：若有**印比帮身**（印生身、比劫助身）则能担财，格局反转为富。
- **身旺财旺**：能担财，主富。**反面**：身旺而无财官食伤泄耗，反主孤独贫乏（身旺无依）。
- **官杀混杂**：正官七杀同现，主事多变、压力、是非。**例外**：身旺用杀、伤官制杀、印星化杀，反成贵格（"杀印相生""伤官驾杀"）。
- **伤官见官**：伤官与正官相克，主口舌波折。**例外**：《滴天髓》明示**金水伤官、木火伤官**等调候所需之官，或有财通关、有印护身时，伤官见官反显贵——不可一概断凶。
- **印星化官杀**：官杀旺而有印化解，主以学识/技术化解压力，贵格。**反面**：印过重而身已旺，反主滞塞孤克（母慈灭子）。
- **食神生财**：食神与财顺生，主以技艺得财。**反面**：食伤泄身过旺而身弱，反主贫困劳碌（泄气太过）。

### 财印关系详解（结合古籍·避免绝对化）

财与印是身弱命局的核心：印生身护身（用）、财耗身（忌）。财印互动定吉凶，须**分程度、忌绝对化**。

**1. 财坏印（贪财坏印）`[古诀]·子平真诠`**
- 成立：身弱喜印，财旺克印，印星受损。
- 应事：因财损名／学业受阻／母亲健康／靠山失力／为利失义。
- **⚠ 程度区别（避免绝对化）**：
  - 印**虚全坏**（无透无根被财克死）→ 重，应母病／学业损／因财失名；
  - 印**透无根**（透干但地支无根·虚浮）→ 轻-中，母缘在／健康尚可，但靠山不稳（印无根）；
  - 印**有根**（地支有印本气）→ 财难坏印，不成立财坏印，反成财印双美。
- 化解：**比劫制财护印**（比劫克财、护住印星），或以印（技术学识）立身不贪财。`[滴天髓]`「财印交战，须比劫通关制财护印」。

**2. 财多压身的心性（避免绝对化·关键）**
- **财多身弱 ≠ 个人贪财／重财**。财多压身 = "**被动卷入财事**、财缘重但担不住、易被财事拖累"。
- 应事：**被动破财**（被拉合伙／投资失败）、富屋贫人（满盘财非己能任）。
- 断心性：财多身弱者未必重财，反可能"不执着金钱但财事找上门、被动卷入大破财"。**勿断"贪财"**。

**3. 印星：正途 vs 非正途（避免绝对化）**
- 印**有根** = 正途学业／文凭／贵人提携；
- 印**透无根**（虚浮）= 非正途（自费、远程、关系操作），贵人有但力虚。
- 断学业看印虚实：印虚者学历多靠"非正途"，非正经考过。

**4. 财印双美（好格局）`[子平真诠]`**
- 财旺 + 印有根（财不坏印）→ 身能担财、印护身，**富贵双全**倾向。「财喜印护、印不碍财」，两不相碍为佳。

**5. 正向引导（财坏印）**：以比劫（合作）／印（技术学识）制财护印，忌贪财冒进；"因财损名"的风险转化为"以技术立身、不贪财、稳守印星（学业/名声/健康）"。引擎 `bazi_core.js` 的 `caiYinRelation` 自动检测财坏印程度并标注 `notes._财印`。

## 六、大运（每运十年）

### 顺逆规则
- **阳年男 / 阴年女 → 顺排**（从月柱的下一柱起，顺天干地支序）
- **阴年男 / 阳年女 → 逆排**（从月柱的上一柱起，逆天干地支序）

### 起运岁（易错，务必严谨）
从生日数到最近的"节"（非"气"）：顺排数到下一节，逆排数到上一节，**3 天折 1 岁**。

**口径铁律：起运岁一律用虚岁**。精确换算规则：**出生当年即虚岁 1，每过一个农历新年（春节）+1**。

注意"出生后 N 实年"与"虚岁"**并非简单 +1**——取决于出生日距下一个春节的远近：
- 通常出生后第 N 个公历年交运，虚岁 ≈ N+1；
- **若出生在农历年末（接近春节）**，出生后不久即过春节，实年与虚岁差值会变大（可能出现 N 实年 = 虚岁 N+2）。
- 故**务必用专业排盘工具（如问真八字）直接读"X 岁起运 + 交运日期"**，不要手算虚岁换算。

例：命主 2000-08-16 生，逆数到寒露（10/8）得 12.6 天 ≈ 4.2 实年，交运落在 1997 年底～1998 年初；1997 年春节（2/7）后命主虚岁 5，至 1998 年春节（1/28）前仍虚岁 5，故交运时**虚岁 5**，记"5 岁起运"。

**务必用专业排盘工具核对**（如问真八字），避免手算口径/节气时分/计数端点出错。起运岁错了，整条大运分段会整体平移 1 岁、当前大运判断跟着错——但流年逐年分析不受影响（流年按公历干支固定）。

### 大运分析方法
1. 大运天干地支的十神 → 定该十年基调（财运/官运/印运/比劫运/食伤运）。
2. 大运十神是日主**喜用还是忌神** → 喜用运顺、忌神运逆。
3. 大运与四柱的冲合刑害 → 定具体事件。
4. 大运**干支十年全程共同作用**，非割裂：前五年**天干权重更高**、后五年**地支权重更高**，但绝非"前五年不看地支、后五年不看天干"。干支生克、与日主喜忌、与四柱冲合须通盘论。
5. **判大运对"流通/补缺"的补益（四参合）**：原局缺或断链的五行，大运若补之（如原局缺食伤、行食伤火运），非但不凶，反主通关泄秀、技术生财。**忌神运不一定是凶运**——若该忌神恰为原局所需（补缺/调候/通关），反化为用。故大运吉凶须先经四参合喜忌（非纯旺衰）再断。

### 大运详断模板（逐运分析清单）

对每步大运按此清单推演（干支→十神→四参合喜忌→流通→逐年应事）：
1. **大运干支 → 十神**：天干十神、地支本气十神（相对日干）。
2. **四参合喜忌定位**：该十神在「旺衰+格局+调候+流通」下是喜是忌？（缺补/调候/通关所需之忌神→化为用）
3. **流通判断**：大运五行是否补原局断链/缺五行？补则通关顺生（如原局木→土缺火，火运通木→火→土，食伤生财）。
4. **基调顺逆**：喜用运顺、忌神运逆（用四参合修正后的喜忌）。
5. **逐年应事维度**：该运十年内逐年看 流年十神 × 大运基调 → 财/官/印/食伤/健康 等维度吉凶。
6. **干支并重**：前五干重、后五支重，通盘论。

## 七、流年应吉应凶（八字最硬的依据）

**核心逻辑：流年十神是日主喜用神 → 该年应吉；是忌神 → 该年应凶。** 吉凶方向是确定的，这是八字推流年最有力的工具。

**⚠ 前提：此处"喜用/忌神"必须是经第五节「四参合」修正后的真用神，而非纯旺衰二元喜忌**。否则会误判——如原局缺食伤者，食伤流年本是补缺泄秀通关之喜，却被旺衰误作忌神年判凶（典型：身弱财多者行食伤火运，靠技术生财得吉，非凶）。

分析步骤：
1. 流年天干地支 → 转十神（相对日干）。
2. 判喜忌（身弱喜印比、身旺喜财官食伤）。
3. 喜用年 → 应学业、贵人、晋升、得财、姻缘（看具体十神）之吉。
4. 忌神年 → 应破财、压力、伤病、口舌、感情波折之凶。
5. 结合大运：大运喜用＋流年喜用＝大吉；大运忌神＋流年忌神＝大凶；一喜一忌＝吉凶参半。

**各十神喜用年应事**：
- 印（喜用）：学业、文凭、长辈贵人、安顿、置业
- 比劫（喜用）：朋友合伙、助身担财、合作得力
- 食伤（喜用）：才华施展、技艺、子女、表达得利
- 财（喜用）：得财、置业、男命姻缘
- 官杀（喜用）：升职、掌权、地位、名声、女命姻缘

## 九、定时辰（用八字反推）

时辰不同 → 时柱不同 → 可能导致：
1. **日主旺衰逆转**（如甲戌时无水印根身弱 vs 乙亥时有亥水印根身旺）
2. **喜用神完全反转**（身弱喜印比 ↔ 身旺喜财官食伤）
3. **同一年吉凶方向相反**

**反推法**：对每个候选时辰，分别推算"某关键流年是喜用年（应吉）还是忌神年（应凶）"，与命主真实事件对照——方向一致则该时辰成立，矛盾则排除。多组方向性验证同时成立即为该时辰。

最强应事维度（八字视角）：
- 父亲：偏财父星，忌财年应父凶（如手术刑伤）
- 学业：印星，喜印年应学业之吉（如得文凭）
- 性格职业：用印＝技术内敛型，用食伤＝口才外向型
- 健康：日主受克耗程度（如甲木受酉金克、土耗＝神经/肝胆弱）
- 财运：财多身弱者强行求财必破（验创业投资成败）
- **婚姻**：男命财（妻）喜忌——喜财年应姻缘之吉、忌财年应婚变，是定时辰高效应事维度
- **格局高低**：两时辰日主旺衰若逆转，身旺能担财官 vs 身弱富屋贫人，人生层次差异显著，可据命主实际成就层次反推
- **早年灾病**：日主受克耗程度不同，幼年体质/灾厄差异明显

## 十、推演步骤

1. **排四柱**：年月日时，列天干、地支、藏干、十神、纳音。
2. **五行力量统计**：数各五行（含藏干）多寡，标旺衰。
3. **定日主旺衰**：三得（令/地/势）综合判断身强身弱。
4. **定喜用忌神**：身弱喜印比忌财官食伤；身旺反之。
5. **定格局**：财多身弱/身旺财旺/官杀混杂等。
6. **排大运**：顺逆＋起运岁，标各运十神与喜忌。
7. **六维归纳**：事业/财/婚/健康/学业/六亲，每维给结论＋十神依据。
8. **流年逐年**：用喜忌应吉应凶推演，验算过往＋预测未来，标喜用年（机遇）与忌神年（防范）。
9. **综合建议**：扬长（用喜用）避短（忌忌神），针对性化解。
10. **免责声明**。

## 十五、正向引导·趋吉避凶（命理产出必守）

命理的本质是趋吉避凶、给人希望与出路，**不是堆砌凶象吓人**。吉凶按原理如实推演，但解析必须给出**转化路径**：

1. **凶运凶年 → 指出"往哪发力"**：婚姻感情凶→转攻事业/财源/学业（先立业后成家）；财运凶→深耕技术、储蓄韬光、忌投机；事业官杀凶→以印（技术学识）化杀、专业立身；健康凶→养生体检、放慢节奏。
2. **忌神运 → 为喜用运蓄力**：忌神运是磨炼沉淀期，把精力投向喜用方向（如本局喜印比食伤→学技术、攒人脉、磨专业），待喜用运到即可发力。
3. **禁绝对化、禁凶吓表述**：用"需注意/宜防守/此运宜 X"替代"大凶/必破/死局"；**每个凶都要配一条出路**。
4. **凶中寻吉、吉中防忌**：给完整图景而非单边恐吓——再凶的运也有可取之维（如财运凶时学业/技术维仍可进），再吉的运也有需防之处。

**凶运转化速查**：
- 财运凶（财忌）→ 宜学技术/深耕专业/储蓄，忌投机合伙。
- 官杀凶（官杀忌）→ 宜以技术学识化杀、积累实力，忌硬刚出头。
- 食伤凶（身已弱）→ 宜节制输出养精力，忌过度透支。
- 婚姻感情凶 → 宜转攻事业财运学业，提升自身，先立业后成家。
- 健康凶 → 宜养生、体检、规律作息、放慢。
- 整体凶运 → 韬光养晦、学习蓄力，为下步喜用运做准备。

## 十六、害刑应六亲（定盘/应事核心·流年触发动六亲星所在支）

> 事件应期须**原局（体）+ 大运（十年环境）+ 流年（当年触发）三层引动**方应，**禁单看原局判事件**。八字层面，六亲刑伤事件常由**流年害/刑动六亲星所在支（财库/印根）**触发。完整方法论（含定盘优先级、教训案例、紫微对照）见 `methods/dingpan-logic.md`。

### 六亲星定位 `[项目验证]`（八字通识基础上的项目实操细化）

- **父亲 = 偏财**（男命通用），看其天干透出与地支藏根/库。父星常藏于**财库**——若偏财为戊土，则藏辰戌（戊土为辰戌本气）。
- **母亲 = 正印**（兼看偏印），看其虚实有无根（印根有无是学业/技术/扛财的根本，见 `dingpan-logic.md` 第一节）。
- 男命妻=正财、子女=官杀（传统）或食伤（任铁樵派）、手足=比劫；女命夫=正官、子女=食伤。

### 害刑动六亲星所在支 → 六亲刑伤 `[项目验证]`

- **害（穿）**：子未、丑午、寅巳、卯辰、申亥、酉戌六组。流年地支**害**六亲星所在支（尤其害动财库/印根）→ 该六亲刑伤。
- **刑**：寅巳申三刑、丑戌未三刑、子卯相刑、辰辰/午午/酉酉/亥亥自刑。流年地支**刑**六亲星所在支，或**自刑**加重该支五行 → 该六亲刑伤。
- **组合**：流年害/刑同时动多个该六亲星所在支（×2 或 ×3）→ 刑伤烈度倍增。
- **十神叠加**：害刑六亲星所在支的流年地支，其十神（相对日干）性质决定刑伤类型——**官害财（父）= 官鬼刑伤父星**、印刑身 = 母缘/自身受累。

### 案例（项目验证·2017 父亲砂轮片手术）

命主（甲日主）偏财**戊土 = 父星**，藏**辰戌**（财库藏戊父星）；原局三戌 = 财库（藏戊父星 ×2）。**2017 丁酉流年**：
- **酉（正官）害戌（财库/父星）×2**（原局两戌被酉害，害动父星所在财库 ×2）；
- **酉酉自刑**（流年酉与原局/大运酉自刑，加重官鬼金气）；
- 十神叠加：酉为甲之**正官（官鬼）** → **官害财（父）+ 自刑 → 父亲刑伤应手术**。

**即：原局父星（偏财戊土）藏财库本无大碍，但流年害刑动其所在支（财库戌）→ 父亲刑伤应期触发。** 若单看原局偏财旺衰判"父无碍"，会误判 2017 父亲手术。定盘时遇"原局应不了"的六亲事件，**必查流年害刑是否动六亲星所在支**。`[项目验证]`

## 十七、真神取法·官杀取清（可复用取用技法）

> 官杀混杂（正官与七杀同现或邻现）是定用神时的高频难点：到底以官为真、还是以杀为真？本节给出**可复用于任何官杀混杂命局**的取清法则——不止命主个案。本节为 `[项目验证]` 抽取自命主乙亥·甲日主定盘辨析（命理师指正"真神是酉金辛正官，非庚金七杀"），与第五节「四参合定用」、第七节「流年应吉应凶」互补。

### 核心法则 `[项目验证]`（八字通识基础上的项目实操细化）

1. **官杀混杂时，取命局本有之官（原局藏/透的官星）= 真神（真官）；去运过之杀（只在大运、原局无的杀）= 非真**。
   - **原局有 = 命主本有禀赋（真）**：原局藏干或透干所带的官星，是命主先天禀赋、终生主导，故为真神。
   - **原局无只运来 = 运过即去（不主导命局）**：仅大运过境、原局并无的杀，是阶段性外力，运过即去，不主导命局层次与事业路线，故非真。
   - **取清逻辑**：真神既定，论事业/官运即以真官为据，去其杀（不取杀为主线）。
2. **论事业/官运用真官，不用运来杀**。事业是终生主线，须看本有禀赋（真官）；运来杀只主该十年阶段性压力/变动，不主事业层次与路线。
3. **官空填实**：真官若旬空（尤其官禄空亡、年支官星空），主**职位虚 / 实利打折**（名义到、实利不随——如升职不加薪、头衔到而权责/薪酬未随）；待**流年官星到（同字填实）则官显威**（真官之利方落实）。
4. **事业路线选择**：
   - **真官（正官）路线** → 正统职位 / 规整晋升 / 专业规整（适合体制、大厂架构、专业序列）；
   - **若误走七杀路线（创业 / 偏锐 / 开创打江山）易破财**——真官命本无杀之禀赋，强行走杀的模式（自行创业、单打独斗、偏锐开疆）与命主本有禀赋不符，主破财折损。
   - 即：**真官命不适合"七杀打江山"模式**，宜走正官规整晋升路线。

### 案例（命主乙亥·甲日主·`[项目验证]`）

命主甲日主，定盘辨析"真神是辛（酉正官），非庚（七杀）"：

| 星 | 命局有无 | 出处 | 真假 |
|---|---|---|---|
| **辛（正官）** | **原局本有** | 藏**酉**（年支，辛禄酉本气）+ 戌（月/日，中气辛） | **真神（真官）** |
| **庚（七杀）** | **原局无庚** | 仅大运庚申（15-24 岁）过境 | **非真（运过即去）** |

- 故**真神 = 辛（酉正官），去庚（七杀）**。论事业走**正官路线**（架构师 / 规整晋升）。
- **事业路线印证**：命主升职路径"资深开发 → 架构师 → 总监"= 标准正官规整晋升路线；被拉合伙创业（七杀路线）→ 破财 200 万（真官命走七杀打江山模式不吉，与"真官命不适合七杀模式"法则吻合）。
- **官空填实印证**：辛官坐酉，**年支酉旬空**（官禄空亡）→ 2021 升职后不再加薪（官位到、实利打折 / 官空——名义到、实利不随）；**2021 辛丑年**（辛官同字填实到位）→ 升职（官显威，真官之利于流年填实时落实）。

### 适用范围与边界 `[项目验证]`

- **适用**：任何**官杀混杂**命局（无论日主旺衰、无论男女），定事业/官运真用神时均可套用本节四法则取清。
- **边界 / 例外**（须与第五节「四参合」交叉，忌机械）：
  - 若原局**杀有强根且透干、官反虚浮无根**，则真神未必是官——须以**原局根气实者**为真（本节法则的精神是"取命局本有禀赋之实者"，非死板"必取官"）。
  - 若原局成**伤官驾杀 / 杀印相生**等杀格贵格（见第五节"官杀混杂"例外），则杀反为用、非取清对象——此时以成格之杀为真神，不拘于"官真杀伪"之形。
  - 官空填实之"空"须以**真官所在支旬空**为准（非任意空亡），且须结合流年同字填实断应期，忌泛论"官空必凶"。
- **取用次第**：先以本节定真神（官 vs 杀），再纳入第五节四参合（旺衰+格局+调候+流通）定真用神——本节是"官杀取清"的专项技法，不替代四参合，而是四参合在官杀混杂场景下的前置取清步骤。
