# 紫微斗数推演方法论

> 本文件为 `ziwei-bazi-reading` 技能的紫微斗数子方法。**独立体系，不含八字术语**。调用本文件时，遵循 SKILL.md 的核心铁律（严谨、附依据、必加免责声明）。

## 一、所需排盘数据

从用户处获取或核对以下要素：
- 性别、农历/公历生辰、真太阳时
- **五行局**（定起运岁）、**命宫干支**、**身宫**所在宫
- **命主**、**身主**、**来因宫**、**斗君**
- 十二宫各宫主星/辅星/小星/神煞、**生年四化**落点
- 若有宫干自化标注：↓离心自化（本宫天干化出）、↑向心自化（对宫天干化入）

**数据自洽核对**：推演前先验证生年四化落宫与命盘标注一致（如癸年应：破军禄、巨门权、太阴科、贪狼忌），不一致则提醒用户数据可能有误。

> 派别提示：**来因宫**为钦天门 / 北派飞星技法（生年四化的"来源宫"），传统三合派不使用来因宫论断。本文档采用飞星＋钦天视角，若纯三合派排盘可忽略此项。

### 四化象义

| 四化 | 象义 | 一句话 |
|---|---|---|
| 化禄 | 财禄、机会、顺遂、缘起 | 得、多、喜 |
| 化权 | 权柄、掌控、变动、力争 | 势、强、争 |
| 化科 | 名声、贵人、平顺、教化 | 名、贵、稳 |
| 化忌 | 执着、阻碍、变动、收敛、结 | 缺、执、变 |

### 自化与飞化解读（关键：化星「落点」决定性质）
- **飞化**：某宫天干四化落**他宫**星（化出/化入他宫）——主宫位间能量流动、因缘牵动。
- **↓离心自化**：**本宫天干四化，化星落回本宫星**。主"由内而外的主动变化、事难久留"。象义随化类而异，**非全为负面**：
  - 自化禄：自己把福气散出、有也留不住 / 主动布施付出
  - 自化权：自己争取、主动掌控、自我强化（中性偏正）
  - 自化科：自己得名、自我修饰
  - 自化忌：自我内耗、执着、自己造成的阻碍（负面）
- **↑向心自化**：**对宫天干四化，化星落入本宫星**。主"由外而内的被动牵动"，吉凶看化类：
  - 化禄入：外来资源机遇（吉） / 化权入：外来赋予掌控力（吉）
  - 化科入：外来贵人名声（吉） / 化忌入：外来是非压力、被牵累（凶）
- 例：太阳↓忌在福德＝自己想太多、内耗（离心自化忌）；太阴↑权入命＝外出/对外时得掌控力（向心权入）。

## 五、大限（每限十年）

### 顺逆规则
- **阳年男 / 阴年女 → 顺行**（命→父母→福德→田宅→官禄→交友→迁移→疾厄→财帛→子女→夫妻→兄弟）
- **阴年男 / 阳年女 → 逆行**（命→兄弟→夫妻→子女→财帛→疾厄→迁移→交友→官禄→田宅→福德→父母）

### 起运岁（五行局）
水二局2岁 / 木三局3岁 / **金四局4岁** / 土五局5岁 / 火六局6岁。每限管10年。

### 大限分析方法
1. 大限命宫主星＋煞星 → 定该十年基调（开创/守成/动荡）
2. 大限四化（大限宫天干化出）落哪宫 → 该十年重点牵动领域
3. 大限与生年四化、本命宫位叠合 → 看叠加吉凶
4. **叠宫**：大限宫＝本命某宫，力量倍增；大限忌冲本命命宫/身宫，该限多事

## 六、流年（逐年）

- **流年命宫**：流年地支落在哪个宫位，该宫即流年命宫。
- **流年四化**：流年天干按四化表化出，落流年命盘各宫，主该年事件。
- **流禄/流权/流科/流忌**入本命或大限重要宫位 → 该年该领域显著事件。
- 流年化忌入流年命宫/本命命宫/大限命宫 → 该年多变故，提前防范。
- 流年化禄入财帛/官禄/大限命宫 → 该年财事业机遇。
